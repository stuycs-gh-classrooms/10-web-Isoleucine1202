Fractal I: The Koch Curve

The Koch Curve is a fractal with a straight line adorned by repeating patterns, growing smaller with depth.

Here is an image of draw_koch(t, 4, 200):

This fractal has been modified such that every other line drawn is slightly shorter than the rest.

Here is my modified fractal's code and an image of draw_koch(t, 4, 200):

        
def draw_koch(t, depth, length):
    if depth == 1:
        t.forward(length)
    else:
        draw_koch(t, depth-1, length/2)
        t.left(60)
        draw_koch(t, depth-1, length/3)
        t.right(120)
        draw_koch(t, depth-1, length/2)
        t.left(60)
        draw_koch(t, depth-1, length/3)


Fractal II: The Sierpinski Triangle

The Sierpinski Triangle is a self-similar fractal with an overall shape of an equilateral triangle, but subdivided into many equilateral triangles.

Here is an image of draw_sierpinski(t, 5, 400):

I modified the triangle by having the pen size gradually increase as you go up and to the right in the triangle.

Here is my modified fractal's code and an image of draw_sierpinski(t, 5, 400, 1.0):

        
pens = 0.03

def draw_sierpinski(t, depth, length, pen_size):
    global pens
    pens += 0.002
    if depth == 0:
        for _ in range(3):
            t.forward(length)
            t.left(120)
    else:
        t.pensize(pen_size)
        draw_sierpinski(t, depth-1, length/2, pen_size + pens)
        t.forward(length/2)
        draw_sierpinski(t, depth-1, length/2, pen_size + pens)
        t.backward(length/2)
        t.left(60)
        t.forward(length/2)
        t.right(60)
        draw_sierpinski(t, depth-1, length/2, pen_size + pens)
        t.left(60)
        t.backward(length/2)
        t.right(60)


Fractal III: The Fractal Tree

The Fractal Tree is a fractal where 'tree branches' split off systematically.

Here is an image of tree(t, 4, 100, 30):

I modified this tree in several ways: First, I altered the color to firebrick. Then, I randomized the lengths of each branch. Finally, I randomized the angles between branches.

Here is my modified fractal's code and an image of tree(t, 4, 100, 30, 3)- p.s. keep scrolling!:

        
import random

def tree(t, depth, length, angle, width):
    t.color('firebrick')
    if depth == 0:
        return
    else:
        t.pensize(width)
        t.forward(length)
        t.left(angle)
        tree(t, depth-1, length*(random.random() + 0.3), angle * (random.random() + 0.43), width - 0.2)
        t.right(angle*2)
        tree(t, depth-1, length*(random.random() + 0.3), angle * (random.random() + 0.40), width - 0.2)
        t.left(angle)
        t.backward(length)
